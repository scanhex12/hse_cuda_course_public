# Tree — высоты вершин дерева

Дано ориентированное дерево из `num_vertexes` вершин. Вершины пронумерованы от 0 до `num_vertexes - 1`. Массив `parents[i]` содержит номер родителя вершины `i`, для корня `parents[0] == -1`.

Реализуйте в `tree.cu` функцию **`findHeights`**:

- **Вход**: `parents` — массив предков на хосте, `num_vertexes` — число вершин.
- **Выход**: массив высот на хосте (выделенный через `malloc`), вызывающий код освобождает его через `free`/`delete[]`.

**Высота вершины** — расстояние до корня (число рёбер на пути). Корень имеет высоту 0.
