# Binary Image — расстояние до ближайшей единицы

Дана бинарная матрица размера N×M (значения 0 или 1). Для каждой ячейки вычислите **манхэттенское расстояние** до ближайшей ячейки в этой же колонке или строчке со значением 1. Если в ячейке уже стоит 1, расстояние равно 0. Если единиц в матрице нет (в столбце или строке нет достижимых единиц), используйте -1.

Реализуйте в `binary_image.cu`:

1. **`findNearestOnesKernel`** — `__global__` ядро: для каждой ячейки `(i, j)` находит минимальное манхэттенское расстояние до ячейки в том столбце или той же строчке со значением 1. Манхэттенское расстояние: `|i1 - i2| + |j1 - j2|`.

2. **`findNearestOnes`** — хост-функция: выделяет память на устройстве, запускает ядро, возвращает матрицу-результат на устройстве.

3. **`solve`** — хост-функция: принимает матрицу на хосте, копирует на устройство, вызывает `findNearestOnes`, копирует результат обратно на хост и возвращает его. Освобождает временную память.

Рекомендуется реализовать вспомогательные функции `allocMatrixHost`, `allocMatrixDevice`, `copyMatrixHostToDevice`, `copyMatrixDeviceToHost`, `freeMatrixHost`, `freeMatrixDevice`.
